🎢 오르락내리락 – 5분 단타 코인 투자 게임
오르락내리락은 5분 동안 100만원을 가지고 코인 단타 투자에 도전하는 iOS 게임 앱입니다.
짧은 시간 내 최대 수익률을 달성해 투자 극락과 나락을 체험할 수 있습니다.

🧰 기술 스택
플랫폼: iOS
UI: SwiftUI
데이터 관리: SwiftData
언어: Swift

🗂️ 주요 기능
게임 시작 시 100만원 지급
실시간 코인 가격 변동을 바탕으로 5분 동안 수익률 극대화 목표


💰 투자 결과에 따른 극락과 나락 체감
수익이 많으면 극락, 손해 시 나락을 경험
자신의 투자 기록을 직접 세울 수 있음


📊 투자 기록 관리
플레이어의 수익률과 투자 결과를 저장
나만의 최고 기록을 확인하며 투자 전략 개선 가능
